# Django_Project
```
This is new Django project named “songcrud” and an app in the project called “musicapp”. 
this repo contains a requirements.txt file housing all the pinned dependencies 
for this  project.

In the models.py file inside the “musicapp” application includes the following Models and Attributes.

Model: Artiste, Song, Lyric
Attributes for “Artiste” : first_name, last_name, age
Attributes for “Song” : title, date released, likes, artiste_id
Attributes for “Lyric”: content, song_id

```
